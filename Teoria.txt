Azure Service Bus
	É um modo de enviar uma mensagem de um ponto a para o ponto b

	Terminologia
		Mensagem: É um objeto json que contém toda a informação que deseja ser enviada
		Queue(fila): É uma lista de mensagens armazenadas em uma determinada ordem,
		geralmente na ordem em que são inseridas(FIFO), para que possam ser recuperadas
		posteriormente.

Pacotes necessários
	Microsoft.Azure.ServiceBus

Criação da fila
	Logar no Azure e buscar pelo serviço Service Bus
	No menu esquerda procurar pela opção Queues e clicar
	Procurar o ícone de criar uma nova fila.
	Após isso dê um nome para a fila e as demais configurações pode manter o que já estiver

Connection String da fila
	Após a criação da fila é necessário criar uma connection string para a mesma, para isso
	acesse no menu esquerdo:
		Shared access policies -> Add
		Em policy name defina um nome. Após isso defina as permissões de acesso à fila
		referentes à connection string recém criada.
		Após os passos anteriores, clique em Create.
		
		Para acessar a connection string, basta clicar sobre a policy recém criada
		e copiar uma das policies disponíveis.
